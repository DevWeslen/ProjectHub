# ProjectHub – Portfólio Acadêmico e Profissional

Este repositório contém o **ProjectHub**, um portfólio web desenvolvido para centralizar e apresentar minha trajetória **acadêmica, científica e profissional**, reunindo **projetos, artigos científicos, certificados e diplomas** em um único ambiente.

O objetivo do sistema é servir como **vitrine técnica**, **acadêmica** e **institucional**, facilitando a avaliação por recrutadores, orientadores, bancas acadêmicas e parceiros profissionais.

---

## 🎯 Objetivos do Projeto

- Centralizar informações acadêmicas e profissionais
- Apresentar projetos de tecnologia de forma organizada
- Disponibilizar artigos científicos publicados
- Facilitar acesso a certificados e diplomas
- Servir como portfólio público online
- Aplicar boas práticas de organização, responsividade e UX

---

## 🧩 Funcionalidades

### 📌 Página Inicial
- Apresentação pessoal
- Links para currículo PDF, LinkedIn e Currículo Lattes
- Ícones das principais stacks tecnológicas
- Texto dinâmico com efeito de digitação
- Navegação para áreas acadêmicas

### 🎓 Diplomas
- Listagem de formações acadêmicas
- Acesso direto aos documentos

### 📄 Certificados
- Organização por curso/evento
- Informações de validade
- Visualização de certificados em PDF via modal

### 📚 Artigos Científicos
- Artigos nacionais e internacionais
- Identificação do tipo de publicação
- Acesso direto aos arquivos ou links oficiais

### 💻 Projetos
- Cards dinâmicos gerados via JavaScript
- Descrição técnica dos projetos
- Tecnologias utilizadas
- Estrutura preparada para integração com GitHub
